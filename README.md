# Simple Calculator

This is a basic calculator web application built using HTML, CSS, and JavaScript. The calculator allows users to perform arithmetic operations such as addition, subtraction, multiplication, and division.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- All Clear (AC)
- Delete (DEL)
- Equals (=)
